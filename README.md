# .github

Organization-wide community health files for the **nics-dp** organization.

Files placed here serve as defaults for every repository in the organization
that does not provide its own copy. See GitHub's documentation on
[default community health files](https://docs.github.com/communities/setting-up-your-project-for-healthy-contributions/creating-a-default-community-health-file).

## Contents

- [`SECURITY.md`](SECURITY.md) - organization-wide security policy and
  vulnerability disclosure process.

> Note: default community health files in this **public** repository apply to
> the organization's public repositories. To provide defaults for private
> repositories, add the same files to a private repository named
> `.github-private`.
